| :--- | :--- |

| 贰/二 nng7\(白\)/ji7\(文\) | 叁/三 sann\(白\)/sam\(文\) |
| 肆/四 si3\(白\)/su3\(文\) | 伍/五 goo7\(白\) / ngoo2\(文\) |

{% hint style="info" %}
「一」 兮白讀音訓自 「蜀」， 是 「獨自，單一」 兮意思；「二」 兮白讀音訓自 「兩」，是 「成雙」 兮意思。
{% endhint %}
